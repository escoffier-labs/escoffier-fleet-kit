## [0.1.0] - 2026-06-27

### Added
- Shared OG/theme tooling for the Escoffier Labs site fleet, including routine version sync and best-effort chat ping after changed syncs.
- Content-aware specials board refresh flow through review PRs.
- Public publishing-status tracking for fleet repos.
- CI lint coverage for shell and `.mjs` scripts.
- Open-sourced under MIT: `LICENSE`, plus `CONTRIBUTING.md`, `SECURITY.md`, and
  `CODE_OF_CONDUCT.md`.
- README now shows a sample OG card (`docs/og-card-sample.png`) rendered by the
  kit, as a proof of the shared dark-ledger card theme.
- Registered the kit's own site in `og/sites.json` so the renderer produces its card.
